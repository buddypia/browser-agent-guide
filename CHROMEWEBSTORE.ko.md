# Chrome Web Store Listing — Browser Agent Guide (한국어)

> 최종 업데이트일: 2026-06-18

## 스토어 등록 정보

**확장 프로그램 이름**
Browser Agent Guide


**한 줄 설명**
브라우저 조작 AI를 위한 안전 레일: 미리 정의된 안전한 액션만으로 페이지를 작동하는 사이드바 AI 가이드.


**상세 설명**
Browser Agent Guide는 브라우저 조작 AI 에이전트가 안전하고 예측 가능하게 자동화를 수행할 수 있도록 돕는 강력한 안전 가이드 및 레일 확장 프로그램입니다.

특징
• 시각적 보완 — 페이지 요소를 클릭하는 것만으로 메모, 마커, 라벨을 간편하게 배치할 수 있습니다.
• 캔버스 드로잉 — 화면 위의 요소에 원, 사각형, 화살표, 자유형 펜으로 직접 그리고, 스크롤이나 레이아웃 리플로우 시에도 요소를 따라 자동으로 스크롤 및 크기가 조정되는 드로잉 주석을 생성합니다.
• 다중 시그널 고정밀 앵커 — 페이지 재방문 시에도 고유 경로 ID, test ID, ARIA 라벨, 텍스트 정보 등을 결합한 다중 신호를 이용해 대상을 확실하게 매번 찾습니다.
• 등록형 액션 제한 (Verb Registry) — AI의 동작 범위를 미리 정의된 안전한 명령어(클릭, 입력, 스크롤 등)로만 제한해 브라우저 상의 예기치 않은 파괴적인 DOM 조작을 차단합니다.
• 사이드바 UI — 대화식 창에서 AI와 직접 명령을 주고받고 실행 이력을 확인하며, 페이지의 상태를 인라인으로 관리합니다.
• 커스텀 스크립트 자동 저장 규칙 — 적용된 CSS와 JS를 크롬의 User Scripts API를 활용해 저장하고 페이지 재방문 시 자동으로 재적용합니다.
• AI 프로바이더 직접 연결 — OpenAI, Anthropic, Gemini 또는 사용자 지정 OpenAI 호환 엔드포인트로 로컬에서 직접 HTTPS 통신을 수행하여 API 키가 외부 서버로 유출되지 않습니다.

사용 방법
1. 툴바에서 Browser Agent Guide 아이콘을 클릭하여 사이드바 채팅 창을 엽니다.
2. 설정 창(또는 사이드바 우측 상단의 톱니바퀴 아이콘)을 열고 사용할 AI 프로바이더의 API 키와 모델명을 등록합니다.
3. 툴바에서 "보완(補足)" 또는 "그리기(お描き)"를 클릭해 웹페이지 상의 요소나 영역을 마킹합니다.
4. 페이지 컨텍스트를 클립보드에 복사해 외부 AI 챗에 붙여넣거나, 사이드바 채팅창에 직접 명령을 입력합니다.
5. AI가 자연어 명령을 안전한 등록형 액션(Verb)으로 실시간 변환해 실행하는 과정을 확인합니다.

개인정보 처리방침
개인정보 보호를 최우선으로 합니다. 본 확장 프로그램은 사용자의 개인정보나 브라우저 방문 기록을 개발자 서버나 제3자에게 전송하거나 수집하지 않습니다. 모든 설정 정보, API 키, 프롬프트 이력, 적용 규칙은 디바이스 내부 로컬 스토리지(chrome.storage.local)에만 안전하게 보관됩니다.

권한 안내
• "방문하는 웹사이트의 전체 데이터 읽기 및 변경" — 웹페이지 상의 요소를 식별하고 주석 오버레이를 표시하며 안전한 액션을 시뮬레이션하기 위해 필요합니다.
• "다운로드 관리" — 작성된 주석 이미지 패키지와 페이지 컨텍스트 텍스트 파일 등을 로컬 다운로드 폴더에 안전하게 다운로드하기 위해 사용합니다.
• "복사 및 붙여넣기한 데이터 변경" — 페이지 상태 컨텍스트 데이터를 클립보드에 바로 쓸 수 있도록 돕습니다.
• "로컬 장치에 데이터 저장" — API 키, 환경 설정, 사이트 기억 규칙을 지속적으로 로컬 장치에 보관하기 위해 필요합니다.
• "백그라운드에서 스크립트 실행" — 메시지 라우팅 및 외부 AI API와의 직접 통신을 지원하는 서비스 워커 구동에 사용됩니다.
• "사이드 패널 레이아웃 제어" — 브라우저 창 우측에 사이드바 채팅 가이드 UI를 정상 표시하기 위해 요구됩니다.
