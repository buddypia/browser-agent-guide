# Chrome Web Store Listing — Browser Agent Guide (简体中文)

> 最后更新日期: 2026-06-18

## 商店上架信息

**扩展名称**
Browser Agent Guide


**一句话简介**
浏览器操作 AI 的安全防护栏：仅通过预定义的安全动作注册表控制任何页面的侧边栏 AI 助手。


**详细描述**
Browser Agent Guide 是一款为浏览器操作 AI 智能体（Agent）提供安全防护栏与辅助能力的 Chrome (Manifest V3) 扩展程序，旨在让非技术人员也能轻松编写可复现、安全且确定性的网页自动化流程。

特点
• 视觉注解 — 点击页面元素即可添加说明、标记与标签。
• 画布绘制 — 可以在页面元素上直接绘制圆圈、矩形、箭头或自由手写，线条随滚动及布局自动缩放与贴合。
• 多信号高精度锚定 — 采用页面重解析技术，结合稳定路径 ID、test ID、ARIA 标签等多重信号，保证重新访问时能百分百定位同一元素。
• 闭环动作注册表 (Verb Registry) — 将 AI 操作限制在预设的安全指令（如点击、填充、滚动等）内，防止 AI 产生非预期的 DOM 注入或越权操作。
• 侧边栏 UI — 支持与侧边栏 AI 聊天以执行指令、查看执行历史并直接管理已保存的页面状态。
• 自适应脚本自动保存 — 自动将配置的 CSS 或 JS 脚本通过 Chrome 的 User Scripts API 保存并在再次访问时自动应用。
• 直连 AI 模型 — 使用本地 API 密钥，直接通过 HTTPS 请求连接 OpenAI、Anthropic、Gemini 或自定义 API 端点，无中间代理服务器。

使用方法
1. 点击浏览器工具栏中的 Browser Agent Guide 图标以打开侧边栏聊天。
2. 打开“设置”页面，输入您本地的 API 密钥并选择您的 AI 模型提供商。
3. 点击“补足”或“画图”按钮来标记网页上的特定元素或区域。
4. 将上下文复制并粘贴至外部 AI，或直接在侧边栏聊天中以自然语言输入指令（例如：“帮我高亮搜索框”）。
5. 监控 AI 动作，查看它如何将自然语言转化为安全的注册指令并逐步执行。

隐私政策
隐私至上。Browser Agent Guide 不会向开发者服务器或任何第三方收集、存储或传输您的个人数据或浏览历史。所有 API 密钥、配置项、对话历史与页面应用规则均保存在您本地设备的本地存储（chrome.storage.local）中。

权限说明
• “读取并更改您在所有网站上的所有数据” — 用于定位页面元素、绘制页面标注图层并安全地执行自动化动作。
• “管理您的下载” — 用于将导出的标注数据包与日志文件下载到本地文件夹。
• “修改您复制和粘贴的数据” — 用于将页面状态上下文写入剪贴板。
• “在本地设备上存储数据” — 用于持久化 API 密钥、个人偏好和网页应用规则。
• “在后台运行脚本” — 用于运行 Service Worker 进行消息路由及直接的 AI API 通信。
• “控制侧边栏布局” — 用于在网页侧边展示 AI 交互和配置面板。
