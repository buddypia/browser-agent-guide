Browser Agent Guide 設定

AI接続、保存範囲、ページごとの記憶ルールを管理します。

AI 接続

APIキーは chrome.storage.local にのみ保存され、外部送信は選択したAI APIへのリクエスト時のみです。

自動保存

サイドバーのチャットで injectHtml / injectCss / injectScript などが成功したとき、 ここで選んだ範囲の再訪時ルールに自動保存されます。個別ルールは下の一覧で後から変更できます。

視覚フィードバック デーモン

「お描きを画像でAIへ」を、ダウンロード保存ではなく常駐デーモンへ WebSocket で直接送ります（有効時）。 デーモンを起動し（cd daemon && npm start）、起動時に表示される token を下に貼ってください。 無効時や未到達時は従来どおり chrome.downloads に保存します。

トークンはローカル WebSocket の認証用です（悪性Webページからの localhost 接続を防ぐ / CVE-2025-52882 対策）。 chrome.storage.local にのみ保存されます。

自動同期はデーモン到達時だけ送信し、失敗時に自動ダウンロードは行いません。

保存先フォルダはデーモン有効時のみ有効です。空欄ならOSの既定ダウンロードフォルダに保存します（安全のためホーム配下のみ採用。ホーム外は daemon の --inbox で指定）。

記憶したURL / 有効化ルール

補足を保存したり、チャットでページに変更を加えたりすると、そのURLのルールが自動で追加されます。 手動追加はドメイン全体など広い範囲に適用したい場合に使います。

有効ラベル一致種別パターン操作

記憶ルール

選択したURL/ルールのページ読込時に毎回適用される動詞アクションのJSON配列です。 チャットで追加された表示変更もここに保存されます。
