# Chrome Web Store Listing — Browser Agent Guide (日本語)

> 最終更新日: 2026-06-18

## ストア掲載情報

**拡張機能の名前**
Browser Agent Guide


**短い説明**
ブラウザ操作AIのための安全レール：安全な登録アクションのみでページを動かすサイドパネルAIガイド。


**詳細な説明**
Browser Agent Guideは、ブラウザ操作を行うAIエージェント向けに、安全で予測可能かつ確実な自動化を提供する拡張機能です。

特徴
• ビジュアル補足 — ページ要素をクリックして、簡単にメモ、目印、ラベルを配置できます。
• キャンバス描画 — 画面上の要素を直接丸や四角、矢印、手書きペンで囲み、スクロールやレイアウト変更に追従する注釈を付与できます。
• 多重シグナルによる高精度アンカー — ページ再訪時でも、安定ID、test ID、ARIAラベル、テキスト情報などを組み合わせた複数シグナルで要素を毎回確実に再解決します。
• 閉じたアクション登録制（Verb Registry） — AIの操作をあらかじめ定義された安全なアクション（クリック、値の入力、スクロール等）のみに制限し、不審なDOM操作を防ぎます。
• サイドパネルUI — チャット形式でAIに直接指示を送り、実行履歴の確認やページのカスタム状態の適用をインラインで管理できます。
• カスタムコード自動保存ルール — 適用したCSSやJSをChromeのUser Scripts APIを利用して保存し、ページ再訪時に自動適用できます。
• AIプロバイダーへの直接接続 — OpenAI、Anthropic、Gemini、またはカスタムOpenAI互換エンドポイントに対して、お手元のAPIキーを使用してローカルから直接HTTPS接続します。

使い方
1. ツールバーのBrowser Agent Guideアイコンをクリックして、サイドパネルを開きます。
2. 設定画面（またはサイドパネル右上の設定アイコン）を開き、利用するAIプロバイダーのAPIキーとモデルを設定します。
3. ツールバーの「補足」または「お描き」をクリックし、ウェブページ上の要素や領域にマークを付けます。
4. ページの文脈（Context）をクリップボードにコピーして外部AIに貼り付けるか、サイドパネルのチャットに日本語で指示（例：「ログインボタンを強調表示して」）を入力します。
5. AIが指示を解析し、安全なアクションに変換されて順次実行される様子を確認します。

プライバシー
プライバシー第一の設計です。Browser Agent Guideは、ユーザーの個人データや閲覧履歴を開発元のサーバーや第三者に収集・送信しません。すべての設定、APIキー、プロンプト履歴、ページ記憶ルールは、お使いのデバイス内のローカルストレージ（chrome.storage.local）にのみ安全に保存されます。

権限について
• 「アクセスしたウェブサイト上にある自分のデータの読み取りと変更」 — ページ上の要素を特定し、注釈を描画し、安全なアクションを実行するために必要です。
• 「ダウンロードの管理」 — ページのコンテキスト、描画した注釈、実行履歴ログをローカルのダウンロードフォルダに保存するために使用します。
• 「コピー＆ペーストしたデータの変更」 — ページの状況を表すコンテキスト文言をクリップボードに書き出すために使用します。
• 「ローカルデバイスへのデータの保存」 — APIキー、設定、およびページ記憶ルールをローカルに永続化するために必要です。
• 「バックグラウンドでのスクリプトの実行」 — メッセージの仲介やAI APIとの通信を担当するサービスワーカーを動作させるために必要です。
• 「サイドパネルのレイアウトの制御」 — ブラウザ画面の横に拡張機能のチャット画面を表示させるために使用します。
